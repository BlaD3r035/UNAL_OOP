# UNAL POO

---

Actividades del curso de Programación Orientada a Objetos (POO)

> Universidad Nacional de Colombia – Sede Medellín

Este repositorio contiene los proyectos y actividades desarrolladas durante el curso, organizados en sub carpetas dentro de un único repositorio.

---

## Estructura del repositorio

Los proyectos se almacenan en sub carpetas, donde cada carpeta corresponde a un proyecto, taller o actividad independiente.

UNAL_POO/

│

├── projects/

│ ├── Actividad_1/

│ ├── Taller_1/

│ ├── Tarea_1/

│ └── ...

├── labs/

│├── lab_1/

│ └── ...

├── README.md

└── .gitignore

Cada sub carpeta contiene el código fuente de un proyecto específico y puede ejecutarse de forma independiente.
Cada proyecto puede contener un README.MD para su uso
